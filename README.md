# Computer Center Booking System

A simple Command Line Interface (CLI) application for students to register, login, and book PC sessions at a computer center.

## Features

- **Student Registration:** Generate a unique student ID based on your surname.
- **Student Login:** Access your dashboard using your student ID.
- **PC Booking:**
  - View PC availability for the week (Monday - Friday).
  - Book up to 3 sessions per week.
  - Limit of 1 booking per day.
- **Booking Management:** View your current bookings and cancel them if needed.

## Prerequisites

- [Node.js](https://nodejs.org/) (version 14.17.0 or higher recommended for `readline/promises` support).

## Installation & Usage

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd <repository-directory>
    ```

2.  **Run the application:**
    ```bash
    node index.js
    ```

No external dependencies are required.
